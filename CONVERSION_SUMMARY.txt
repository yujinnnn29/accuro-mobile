Date: January 2026
Purpose: Convert Accuro website to a complete React Native mobile application

OVERVIEW
================================================================================

The mobile app had a solid foundation with authentication screens complete but
authenticated users saw a placeholder screen. This conversion implemented the
full main app experience for both users and administrators.

BEFORE: PlaceholderMainScreen showing "Main app screens coming soon..."
AFTER:  Complete bottom tab navigation with all user and admin functionality

API SERVICES (3 new files + 2 modified)
---------------------------------------
1. src/api/quotationService.ts (NEW)
   - createQuotation: Submit cart as quotation request
   - getAllQuotations: Admin view all quotations
   - getMyQuotations: User's quotations
   - getQuotation: Single quotation detail
   - approveQuotation: Admin approve with pricing
   - rejectQuotation: Admin reject

   - getNotifications: Fetch all notifications
   - getUnreadCount: Count of unread
